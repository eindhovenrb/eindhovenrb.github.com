# Eindhoven.rb

Eindhoven.rb is a monthly meet-up for Ruby-minded people from around Eindhoven, The Netherlands.

## Compiling and editing the site

    bundle install
    jekyll --server --auto
    open http://localhost:4000

## Deployment

Simply push to github.
